HTML Image

        html에서 이미지를 삽입하려면 img 태그 사용
        img 태그는 end tag가 없는 empty tag이다.
       
        이미지 태그를 사용시 절대 / 상대 경로가 존재한다

        절대 : 어떠한 웹페이지, 파일이 가지고 있는 고유 경로, 최상위 디렉토리를 포함한 주소를 갖는 경로를 말함

        상대 : 최상위 디렉토리를 포함하지않고 현재 위치한 곳 기준으로 해당 파일 위치까지 경로
    
        

        해당 태그를 살펴보면 src와 alt가 보인다.
        
        src : "url" 및 위치 표현
        alt : "대체 텍스트" 해당 src가 틀리거나 에러로 인해 이미지가 보여지지않을 경우 사용자에게 이미지 대신 보여줄 텍스트를 의미

        이미지의 너비와 높이를 조절하는 속성이다.

        width : ""

        height : ""

        
        또한 앞의 a href의 링크를 사용해서 이미지 자체를 링크 형식으로 만들 수 있다.
