Text Formatting


        'Html5'에서 텍스트에 특별한 의미를 부여하기 위한 태그
         b : 텍스트를 굵게

         strong : 텍스트를 굵게 + 중요함 
        
         i : 텍스트 이탤릭체
        
         em : 텍스트 이탤릭체 + 중요함 의미
        
         small : 텍스트의 크기 작게 
        
         sub : 첨자 텍스트
        
         sup : 위첨자 텍스트
        
         ins : 텍스트에 밑줄 
        
         del : 텍스트에 취소선 
        
         mark : 텍스트에 형광펜 강조

Quotation / Comment


         q : quote의 약자, 짧은 문장을 인용할 때 사용하는 태그 ""로 표시가 된다

        
 blockquote : 긴 문장을 인용할 때 쓰이는 태그 브라우저에는 들여쓰기로 표현된다
